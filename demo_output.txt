🏢 实际业务场景示例 (Real Business Examples):

1. 表单输入框 (Form Input Fields):

   • 输入框固定宽度300px，高度40px
   • Fixed width 300px, height 40px for input fields

2. 响应式文本卡片 (Responsive Text Cards):
   card(text("Long content..."))->width(250)
   • 卡片宽度固定250px，高度根据文本内容自适应
   • Card width fixed at 250px, height adapts to text content

3. 统计指标显示 (Statistics Display):
   panel({icon, title, value})->width(180)->height(120)
   • 统计卡片统一尺寸，便于网格排列
   • Uniform card sizes for grid layout

4. 侧边栏导航 (Sidebar Navigation):
   navItem("Home")->width(200)
   • 导航项固定宽度，高度根据文本和图标自适应
   • Fixed width nav items, height adapts to content
